# DBBTI — Dumbass Big Brain Type Indicator

> 你的脑回路，值得被认真研究一下。

一个搞怪但有深度的人格测试。灵感来自 MBTI，但更贴近当代互联网人的精神状态。

本来只是想做个搞笑测试发朋友，结果写着写着就认真了——16 种人格、4 大维度、心理学理论支撑……说好的摸鱼项目怎么变成了论文。

**在线体验**: [dbbti.pages.dev](https://dbbti.pages.dev)

## 四大维度

| 维度 | A极 | B极 | 你可能在想 |
|------|-----|-----|-----------|
| 能量模式 | 疯 (Manic) | 丧 (Depressed) | 我是在发疯还是在发霉？ |
| 社交模式 | 牛 (Extro) | 恐 (Intro) | 我是人群焦点还是人群障碍物？ |
| 驱动模式 | 卷 (Grind) | 躺 (Chill) | 我卷到飞起还是躺到地心？ |
| 表达模式 | 毒 (Savage) | 甜 (Sweet) | 我输出伤害还是输出糖分？ |

## 功能

- 16 道精心设计的沙雕问题
- 16 种人格类型 + 1 个隐藏彩蛋（祝你好运找到它）
- 每种人格附带完整报告：人格概述、超能力 & 盲区、四维度解读、人际关系、职场指南、灵魂拷问……
- 中英双语支持
- 结果图一键保存分享
- 移动端适配

## 技术栈

- React 19 + TypeScript
- Vite
- TailwindCSS v4
- Framer Motion
- html-to-image / canvas-confetti
- Cloudflare Pages

## 本地运行

```bash
npm install
npm run dev
```

## 部署

```bash
npm run build
# dist/ 目录部署到 Cloudflare Pages
```
